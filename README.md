# Audrey Two magnetic locking ring

A parametric 3D-printable system for magnetically securing man-eating pot plants to flower shop counters.

Created by [Whitkirk Arts Guild](https://whitkirkartsguild.com) for their [2026 production of Little Shop of Horrors](https://whitkirkartsguild.com/shows/little-shop-of-horrors-2026).

## "I've never seen anything like it before. What is it?"

It's an OpenSCAD file which will render locking rings we use to fix our Audrey Two plant pot to a flat surface. It has holes through the middle so that the plant operator can reach though and puppet as needed.

## "What do you want from me?"

1. Install [OpenSCAD](https://openscad.org/).
2. Open `a2-mag-ring.scad`.
3. Adjust parameters to customize if needed.
5. Render and export files for 3D printing, and print.
1. Glue magnets of your choice to the locking rings. Make sure to keep everything in the same polarity orientation so you can lock the pot in any position.
